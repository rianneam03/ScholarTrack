# ScholarTrack

**ScholarTrack** is a full-stack student tracking platform built for an **educational nonprofit**. It securely manages student information, attendance, and performance, demonstrating full-stack development skills with a React frontend and Django backend.

---

## Tech Stack

- **Frontend:** React  
- **Backend:** Django, Django REST Framework  
- **Database:** SQL Server / SQLite (local development)  
- **Deployment:** Render (backend)  
- **Environment Variables:** `python-dotenv` for backend secrets, `.env` for frontend API URL  

---

## Features

- User authentication with role-based access  
- Add, edit, and view student records  
- Attendance tracking  
- Secure API endpoints with Django REST Framework  
- Frontend connected to backend via environment-configured API URL  

---

## Live Demo

You can view the app here: [ScholarTrack Live](https://scholartrack-frontend.onrender.com/)  

> The app requires login to access student data. Public users can view the interface, but no real student data is exposed.  

---

## Notes

- No real student data is included in this repo. All sensitive information is managed via environment variables and ignored in Git.  
- Developed as part of a portfolio for demonstrating full-stack development skills.  
- Created specifically for an educational nonprofit to manage student information securely.
